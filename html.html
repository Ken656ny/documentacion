🔥 Guía Completa para Crear un CRUD en Laravel

En esta guía aprenderás a crear un CRUD (Create, Read, Update, Delete) en Laravel paso a paso. Ideal para principiantes que vienen de frameworks como Flask o Django.

1️⃣ Crear un nuevo proyecto Laravel

composer create-project laravel/laravel crud-laravel

Luego entra al proyecto:

cd crud-laravel

Ejecuta el servidor de desarrollo:

php artisan serve

Abre en tu navegador: http://127.0.0.1:8000

2️⃣ Configurar la base de datos

Edita el archivo .env:


DB_CONNECTION=mysql
DB_HOST=127.0.0.1
DB_PORT=3306
DB_DATABASE=crud_laravel
DB_USERNAME=root
DB_PASSWORD=tu_contraseña

Crea la base de datos en MySQL antes de continuar.

3️⃣ Crear la migración y el modelo

php artisan make:model Estudiante -m

Esto crea:

  • app/Models/Estudiante.php
  • database/migrations/xxxx_create_estudiantes_table.php

Editar migración:


public function up(): void
{
    Schema::create('estudiantes', function (Blueprint $table) {
        $table->id();
        $table->string('nombre');
        $table->string('email');
        $table->string('telefono');
        $table->timestamps();
    });
}

Ejecutar migraciones:

php artisan migrate

4️⃣ Crear el controlador

php artisan make:controller EstudianteController --resource

Esto crea un controlador con todos los métodos CRUD:

  • index() → listar estudiantes
  • create() → mostrar formulario
  • store() → guardar estudiante
  • edit() → editar estudiante
  • update() → actualizar datos
  • destroy() → eliminar registro

Ejemplo de método store():


public function store(Request $request)
{
    $request->validate([
        'nombre' => 'required|string|max:255',
        'email' => 'required|email',
        'telefono' => 'required|string|max:20',
    ]);

    \App\Models\Estudiante::create($request->all());

    return redirect()->route('estudiantes.index')
                     ->with('success', 'Estudiante agregado correctamente');
}

5️⃣ Configurar el modelo

Archivo: app/Models/Estudiante.php


namespace App\Models;

use Illuminate\Database\Eloquent\Factories\HasFactory;
use Illuminate\Database\Eloquent\Model;

class Estudiante extends Model
{
    use HasFactory;

    protected $fillable = ['nombre', 'email', 'telefono'];
}

6️⃣ Rutas

Archivo: routes/web.php


use App\Http\Controllers\EstudianteController;
use Illuminate\Support\Facades\Route;

Route::get('/', function () {
    return view('welcome');
});

Route::resource('estudiantes', EstudianteController::class);

7️⃣ Vistas (Blade)

Crea la carpeta resources/views/estudiantes y dentro los siguientes archivos:

index.blade.php


<h1>Listado de Estudiantes</h1>
<a href="{{ route('estudiantes.create') }}">Agregar Nuevo</a>

<table border="1">
    <tr>
        <th>ID</th>
        <th>Nombre</th>
        <th>Email</th>
        <th>Teléfono</th>
        <th>Acciones</th>
    </tr>
    @foreach($estudiantes as $e)
    <tr>
        <td>{{ $e->id }}</td>
        <td>{{ $e->nombre }}</td>
        <td>{{ $e->email }}</td>
        <td>{{ $e->telefono }}</td>
        <td>
            <a href="{{ route('estudiantes.edit', $e->id) }}">Editar</a>
            <form action="{{ route('estudiantes.destroy', $e->id) }}" method="POST" style="display:inline;">
                @csrf
                @method('DELETE')
                <button type="submit">Eliminar</button>
            </form>
        </td>
    </tr>
    @endforeach
</table>

8️⃣ Probar el CRUD

Ejecuta el servidor:

php artisan serve

Abre: http://127.0.0.1:8000/estudiantes

Ya deberías poder crear, editar, listar y eliminar estudiantes 🎉
